Fleet Away

An Airline Reservation System

Description

Fleet Away is a web-based Airline Seat Reservation Portal designed to provide a convenient platform for booking flight tickets. This system uses DBMS technology to manage user data, while the user interface and connectivity are implemented with respective software applications.


Skills Utilized

  • HTML, CSS, PHP(Front-end)
  • MySQL(Back-end)
  • PHP Server(Xampp)

Role: Front-end Developer
